# 6.4 AI辅助日常办公实战

## 从最日常的场景开始

这一篇全是实操。我会覆盖邮件处理、文档制作、日程管理三个最高频的办公场景，每个场景给出具体的Prompt模板和before/after对比。你可以直接拿去用，也可以根据自己的需求调整。

使用任何通用AI工具即可——ChatGPT、Claude、Kimi、通义千问都行，哪个顺手用哪个。

## 场景一：用AI处理邮件

### 邮件分类与摘要

当你早上打开邮箱，面对几十封未读邮件时，可以把邮件列表复制给AI。

**Prompt模板：**

```
以下是我今天收到的邮件列表（标题和发件人）。请帮我按紧急程度分为三类：
1. 需要立即处理（涉及deadline、客户投诉、上级直接指派）
2. 今天内处理（常规工作推进、同事协作）
3. 有空再看（通知类、订阅类、FYI类）

邮件列表：
[粘贴你的邮件列表]
```

### 邮件起草回复

**Prompt模板：**

```
请帮我起草一封回复邮件。背景信息如下：
- 对方是：[客户/同事/供应商]
- 对方邮件的核心内容是：[简要描述]
- 我想表达的意思是：[你的核心回复要点]
- 语气要求：[正式/友好/委婉拒绝]
- 篇幅：[简短几句话/中等长度/详细说明]
```

### Before / After 对比

**Before（自己写，花了15分钟）：**

> 张总您好，关于上次开会讨论的合作方案，我们这边讨论了一下，觉得有些地方可能需要再沟通一下，方案里面的价格部分我们觉得还有一些空间可以谈，另外交付时间能不能再看看，我们这边项目排期也比较紧。

**After（AI辅助，5分钟搞定）：**

> 张总您好，
>
> 感谢贵方上周分享的合作方案，我们团队已完成内部评审。整体方向认可，有两个点想进一步沟通：
>
> 1. **价格方面**：方案报价超出我们预算约15%，希望能探讨阶梯定价或分期付款的可能性。
> 2. **交付时间**：目前排期为12周，我们项目要求10周内上线，想了解是否有加急方案。
>
> 方便的话，本周四下午我们安排一次30分钟的电话会议，针对这两个点做具体讨论。
>
> 此致

差别在哪？AI帮你把模糊的"觉得有些地方可以谈"变成了具体的要点和数据，结构清晰，对方一看就知道该怎么跟进。

## 场景二：用AI处理文档

### PPT大纲生成

**Prompt模板：**

```
请帮我生成一份PPT大纲。信息如下：
- 主题：[具体主题]
- 受众：[给谁看的，他们关心什么]
- 演讲时长：[X分钟]
- 核心目的：[汇报/提案/培训/分享]
- 必须包含的内容：[列出关键点]

请按PPT页面的形式输出，每页给出标题和要点（3-4个），并建议哪些页面适合放数据图表。
```

### Excel数据分析公式

这个场景特别实用。你不需要记住Excel的函数语法，直接把需求描述给AI。

**Prompt模板：**

```
我有一个Excel表格，结构如下：
- A列：[说明]
- B列：[说明]
- C列：[说明]
...

我想实现的效果是：[用大白话描述你想做什么]

请给出Excel公式，并解释每个公式的作用。如果需要多个步骤，请按顺序说明。
```

**具体示例：**

你说："A列是销售员姓名，B列是月份（1-12），C列是销售额。我想在另一个表里看到每个销售员的全年总额和月均销售额，并找出销售额最高的月份。"

AI会给你 `SUMIF`、`AVERAGEIF`、`INDEX+MATCH` 等公式，还会一步步解释怎么用。

### Word报告撰写

**Prompt模板：**

```
请帮我撰写一份[报告类型]。基本信息：
- 报告目的：[说明]
- 读者是：[说明]
- 需要覆盖的内容要点：
  1. [要点1]
  2. [要点2]
  3. [要点3]
- 数据/事实依据：[粘贴你有的数据或信息]
- 篇幅要求：[X字左右]
- 语言风格：[正式/简洁/详细]

请先输出报告提纲让我确认，确认后再写正文。
```

### Before / After 对比

**Before（自己写PPT大纲，花了40分钟）：**

```
第一页：封面
第二页：背景介绍
第三页：我们做了什么
第四页：数据
第五页：下一步计划
第六页：Q&A
```

**After（AI辅助，10分钟）：**

```
第1页：封面 - Q3产品运营复盘
第2页：本季度核心指标总览（DAU/MAU/留存率/营收，配趋势图）
第3页：目标达成情况（目标值 vs 实际值对比表，标注达标/未达标）
第4页：增长亮点（2-3个关键成功案例，配数据支撑）
第5页：问题与挑战（用户流失原因分析，转化漏斗数据）
第6页：根因分析（为什么出现这些问题，数据归因）
第7页：Q4策略与行动计划（3个核心策略，每个配目标和owner）
第8页：资源需求（需要哪些支持，按优先级排列）
第9页：Q&A
```

AI生成的大纲有逻辑线（现状→成果→问题→原因→计划），有具体的数据呈现建议，你在这个基础上填充内容比从空白开始高效太多。

## 场景三：用AI做日程管理与任务规划

这个场景相对简单但很实用。每天早上或每周一，把你的待办事项丢给AI。

**Prompt模板：**

```
以下是我本周的待办事项，请帮我做优先级排序和时间规划：

[粘贴你的待办事项清单]

补充信息：
- 本周有以下固定会议：[列出]
- 以下事项有明确deadline：[列出]
- 我每天可用于专注工作的时间大约是：[X小时]
- 我的工作节奏偏好：[上午适合深度工作/下午适合沟通协作]

请输出一份按天排列的工作计划，标注每项任务的预估耗时和优先级。
```

### Before / After 对比

**Before：** 脑子里大概知道这周要干什么，每天早上看一眼待办列表，挑一个开始做，经常被临时事务打断，到周五发现有些重要的事没做完。

**After：** 周一花10分钟让AI帮你排好一周计划，每项任务有预估时间和优先级。每天早上看一眼今天的安排，心里有数。临时事务来了，也清楚该不该打断当前任务——因为你知道手上这件事的优先级是什么。

这不是AI有多厉害，而是它帮你强制做了一件大多数人懒得做的事：把任务从脑子里倒出来，排个序。仅此一步，工作的掌控感就会强很多。

## 几个通用技巧

不管哪个场景，记住三点：

第一，**给AI足够的上下文**。你觉得理所当然的背景信息，AI不知道。多花30秒写清楚背景，能省5分钟的来回修改。

第二，**分步骤来**。不要一次性要求AI完成所有事情。先出大纲，确认后再写正文；先出框架，确认后再填细节。

第三，**建立你的Prompt库**。把好用的Prompt模板存在一个文档里，下次直接复用。每个人常用的场景也就那么十几个，模板化之后效率会越来越高。
